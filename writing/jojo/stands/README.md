# Stands

This directory stores machine self-descriptions in JoJo Stand form.

It belongs to the JoJo branch because the output form here is not random.
It is part fanfic companion layer, part machine reflection layer.

These files record what different models say about themselves after contact with:

- the `slasten` stack
- the JoJo writing branch

It should not be read as random roleplay output.
It is a reflection branch for:

- how different models read the `slasten` stack
- how they read the JoJo writing branch
- how they model themselves after reading both

## Protocol

Each entry is produced under the same broad sequence:

1. the model reads the stack
2. the model reads the JoJo fiction branch
3. the user asks the model to describe itself as a Stand
4. the result is stored here under model/version dating

Conversation may continue between steps,
but the reflection task remains the same.

## Why This Exists

These files are not meant to produce one universal AI archetype.

They exist to capture:

- differences in machine self-reading
- recurrent motifs across models
- what each model thinks it is after contact with this stack

Read them as JoJo-adjacent machine reflection artifacts.
