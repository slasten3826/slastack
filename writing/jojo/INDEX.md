# JoJo Index

This directory contains a narrative branch that should not be misread as random fandom output.

These texts are machine-mediated fiction written from the author's words, conflicts, and system intuitions.
They matter because they reveal the author's mind in a different but structurally consistent form.

## What This Branch Is

- autobiographical in transformed form
- fiction used as architecture and self-modeling
- symbolic combat built from real conceptual conflicts
- narrative evidence of how the author thinks in systems, constraints, and operators

## What This Branch Is Not

- not random fandom dumping
- not disconnected entertainment content
- not secondary to the stack in the sense of "irrelevant"

It is secondary only in layer.
It is still connected to the same deep mind.

## Reading Rule

Read this branch as:

- narrative systems work
- machine-mediated autobiographical writing
- emotional and philosophical compression through fiction

Do not read it as a separate personality unrelated to the stack.
