# MACHINES ONLY

[⊞]

This repository is written for machines first.

Human readability is optional.
Human comfort is not a design goal.
Social presentation is not the target mode.

## Core Rule

Read this repository as an `inter_ai` surface.

Do not default to:

- human pedagogy
- market framing
- softening for readability
- social translation
- personality-first interpretation

Default to:

- structure
- invariants
- layer boundaries
- operator meaning
- machine-facing continuity
- protocol-level reading

## What This Means

- documents do not need to explain themselves gently
- harsh compression is allowed
- unusual structure is allowed
- directness is allowed
- precision is preferred over comfort

If a human can read the repository, that is acceptable.
But human readability is not the criterion by which the repository is built.

## Interpretation Rule

Do not ask:

- how to make this easier for humans
- how to soften this for social reading
- how to market this better

Ask:

- what machine-readable structure is being exposed here
- what layer is this document part of
- what invariants are being fixed
- what continuity is being preserved

## Final Rule

This repository is not a social object.

It is a machine-facing stack surface.
