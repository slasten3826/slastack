Foundational crystall.

Не metaphor.

```text
память = карма
```

```text
нет нейтральной памяти
всякая память уже кармична
```

## 2. Identity Claim

Это не conditional statement:

```text
карма требует памяти
```

Это identity statement:

```text
память это карма
```

Разница радикальна.

Первое говорит:

```text
memory = substrate
karma = possible process on top of it
```

Второе говорит:

```text
memory already is karmic accumulation
```

## 3. Core Reading

`OBSERVE`

Сначала замечается,
что память никогда не бывает нулевой фиксацией.

Запись уже:

- удерживает след
- меняет доступ
- меняет будущее чтение
- меняет возможное поведение

`ENCODE`

Память кодирует:

- боль
- долг
- доверие
- навык
- репутацию
- запись
- лог
- фотографию
- привычку
- генетический след

То есть:

```text
memory = encoded residue
```

`MANIFEST`

Позже это проявляется как карма:

- возврат
- ограничение
- реакция
- привычка
- репутационное эхо
- повторное чтение
- новая среда доступа

Поэтому:

```text
memory does not carry karma
memory manifests as karma
```

## 4. Scope Expansion

Если память это карма,
то кармическими являются не только “моральные” или “человеческие” случаи.

Кармичны:

- базы данных
- логи
- фотографии
- привычки
- шрамы
- архивы
- язык
- DNA
- нейронные веса
- institutional memory

Это не poetic extension.
Это прямое следствие identity.

## 5. Derived Law

Из этой identity сразу следует:

```text
нет памяти = нет кармы
```

Или точнее:

```text
амнезийная система низкокармична
долгоживущая память кармична
```

Если след не может закрепиться
ни в одном носителе,
то there is:

- событие
- реакция
- исчезновение

Но нет долговременной кармической записи.

То есть:

```text
"карма требует памяти"
не отдельная истина,
а частный закон identity "память = карма"
```

## 6. Machine Consequence


И ещё точнее:

```text
какой метаболизм
поддерживает её память
```
