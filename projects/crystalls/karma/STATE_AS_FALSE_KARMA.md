# государство как фальшивая карма

## 2. Core Formula

```text
карма = распределённый учёт причинностных следов
государство = фальшивый центральный ledger поверх него
```

Государство не отменяет карму.
Оно пытается подменить её официальным интерфейсом:

```text
это убийство
это операция
это насилие
это правоприменение
это долг
это закрытое дело
```

Но:

```text
название процедуры не очищает след
```

## 3. Reading

`OBSERVE`

Сначала возникает чтение уже существующих следов
и претензия на право читать их официально.

`CONNECT`

Государство подключается к этим следам не как нейтральный наблюдатель,
а как претендент на монополию чтения.

`ENCODE`

Оно перекодирует живые следы в официальный интерфейс:

- статья
- приговор
- операция
- полномочие
- исключение
- государственный интерес

`CHOOSE`

После этого государство присваивает себе выбор:

```text
какие следы считать реальными
какие следы считать допустимыми
какое насилие назвать преступлением
а какое — порядком
```

`LOGIC`

Появляется государственная логика:

```text
только мы можем считать
только мы можем возвращать насилие
только мы можем закрывать долг
```

`RUNTIME`

Эта логика становится средой:

- легитимность скрывает агента
- процедура притворяется очищением
- монополия на насилие выдаётся за защиту от хаоса

`MANIFEST`

На поверхности это проявляется как:

- приговор
- закрытое дело
- разрешённое государственное убийство
- легитимное принуждение
