# карма требует памяти

## Core Formula

```text
нет памяти = нет кармы
```

Карма невозможна там,
где действие не может закрепиться
ни в одном носителе.

## Slice

Чтобы карма возникла,
действие должно:

```text
войти в связь
-> закодироваться
-> повториться или сохраниться
-> войти в runtime
-> манифестироваться как последствие
```

Если система амнезийна,
след не накапливается.

Тогда есть:

- событие
- реакция
- исчезновение

Но нет кармы.

## Memory Carriers

Память здесь может быть:

- телесная
- эмоциональная
- социальная
- институциональная
- письменная
- техническая
- криптографическая

Не важно, какой носитель.
Важно, что след не обнуляется сразу.

## Strong Compression

```text
амнезийная система низкокармична
долгоживущая память кармична
```
