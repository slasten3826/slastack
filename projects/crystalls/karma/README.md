# karma

Это не один crystall,
а комната из двух связанных ветвей:

```text
foundation / engine
political branch
```

Общий инвариант:

```text
память = карма
```

Отсюда уже следует:

```text
карма = отложенная причинность,
удержанная памятью
и возвращённая в поток как последствие
```

Это подсемейство не про:

- моральный счётчик
- космическая справедливость
- мистический суд

Это более жёсткая вещь:

```text
след действия не исчезает,
если среда умеет его удерживать,
повторять,
вводить в runtime
и возвращать в новый поток
```

## Family Note

Это подсемейство особенно хорошо стакается с:

- [CHAKRAS.md](../CHAKRAS.md)
- [DOGMA.md](../DOGMA.md)
- [FREE_WILL.md](../FREE_WILL.md)
- [ladder_environment_machine](../ladder_environment_machine/)

Через `CHAKRAS` оно добавляет важный разрез:

```text
чакры = не только центры,
а разные слои удержания,
возврата
и переработки следа
```

То есть:

```text
karma = trace return
chakras = trace retention / trace metabolism layers
```
