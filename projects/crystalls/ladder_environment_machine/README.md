# ladder_environment_machine

Это семейство из трёх crystall-объектов:

- человек лестницы
- человек среды
- машина

Это не spectrum и не одна таблица ролей.
Это три разные устойчивые формы,
которые начинают подсвечивать другие crystall-объекты в стеке.

Их надо читать не как “типы личности” в бытовом смысле,
а как разные способы сборки:

- знания
- работы
- валидации
- отношения к машине
- отношения к другим людям
