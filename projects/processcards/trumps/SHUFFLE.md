# SHUFFLE

Clarified trump-event draft.

This is not locked card wording.

## 1. Core Reading

`‚òµ ENCODE` creates:

- memory
- order
- residue
- future pressure

`‚ò≤ CYCLE` returns what was fixed
back into motion.

Short reading:

```text
encoded residue re-enters circulation
```

Even shorter:

```text
history becomes future again
```

## 2. Trigger

When this trump becomes known,
resolve it immediately as an event.

Possible known-states include:

- drawn into hand
- observed in latent row
- revealed from target compiler
- manifested from hidden state
- otherwise exposed by card effect

## 3. Clarified Effect

On reveal / become known:

1. Take all cards from grave.
2. Shuffle those cards into deck.
3. Shuffle deck.
4. Do not affect trump zone directly.
5. After this event fully resolves, move this trump into trump zone as normal.
6. If this causes the standard third-trump release, resolve that release by the global trump ecology law.

## 4. Minimum Effect

This trump always performs:

```text
shuffle deck
```

So even if grave is empty,
this event still changes future uncertainty.

If grave is non-empty,
it additionally recycles residue into future circulation.

The two operations are not identical:

```text
grave return != deck destabilization
```

Both belong to the event.

## 5. Resulting State

After resolution:

- grave cards are no longer in grave
- grave becomes emptier, possibly empty
- deck contains previous grave cards
- deck order is randomized
- current manifest chain is unchanged
- current hand is unchanged
- target compiler is unchanged
- runtime lane is unchanged
- trump zone is unchanged unless third release is triggered by the standard ecology rule

Short zone formula:

```text
grave -> deck -> shuffle
trump -> trump zone
```

## 6. Why This Works

This event does not create spectacle.
It creates ecological pressure.

It affects:

- deck future
- grave residue economy
- encode stability
- long-cycle planning
- probability pressure

It does not directly affect:

- current visible board
- current hidden board
- immediate target state
- runtime slot

So this trump belongs to the class:

```text
circulation trumps
```

Not:

```text
scene rewrite trumps
```

## 7. Timing Character

Early game:

- low grave recycle
- meaningful uncertainty injection

Mid game:

- noticeable recirculation
- stronger disruption of encoded expectations

Late game:

- heavy fate recomposition
- grave pressure released back into deck

So this trump scales naturally with match history.

## 8. Boundaries

`SHUFFLE` affects:

- grave
- deck
- future probability

`SHUFFLE` does not affect by default:

- manifest row
- latent row
- hand
- target compiler
- runtime lane
- trump zone contents already stored there

This keeps the event clean,
readable,
and globally useful.
