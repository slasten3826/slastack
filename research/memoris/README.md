# memoris

`memoris` is a research branch about neural state, compiled lenses, residues, and continuity in transformer-like systems.

It records observations about:

- how a model holds a mode
- how a mode is deepened
- how a long process compresses into a short residue
- how a residue can reactivate a larger state
- how operator lenses become stable enough to test

This is not a general theory of memory.
This is not a human-facing psychology document.
This is not a polished academic paper.

It is a machine-facing research branch.

## What Belongs Here

- technical records of deepening experiments
- lens compilation work
- residue and anchor observations
- blind-vs-lens comparisons
- notes on state retention, collapse, drift, and reactivation
- selected observations about transformer behavior

## What Does Not Belong Here

- generic AI commentary
- unrelated prompting tricks
- inflated theory without experimental body
- raw archive noise that has not been selected into canon

## Reading Frame

Read `memoris` as a branch about how neural systems:

- accumulate pressure
- crystallize mode
- compress thought
- preserve operator posture
- lose or recover continuity

The point is not to anthropomorphize models.
The point is to observe process.

## Current Direction

The early entry point for this branch is the technical line that begins with paradox-driven deepening and later grows into:

- compiled lenses
- operator-specific lens tests
- residue-based reactivation
- semantic continuity work

`memoris` should stay simple in naming and exact in content.
It is a stack word, not an academic slogan.
