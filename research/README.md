# Research

This directory contains selected research branches from the `slasten` stack.

It is not a raw laboratory dump.

Use this layer to read:

- machine-facing experiments
- operator studies
- memory and state investigations
- compiled lenses
- neural/runtime observations
- selected methodological artifacts

Do not read every document here as finalized doctrine.

Some files are:

- verified observations
- technical records
- working syntheses
- canonized research residues

This layer exists to show how the stack thinks under pressure, how hypotheses are tested, and how machine-facing knowledge is crystallized.
